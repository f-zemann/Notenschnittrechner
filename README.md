# Notenrechner (Python)

Kleines Konsolenprogramm zur Erfassung von Schulnoten (1–6) und
Berechnung des Durchschnitts.

## Lernziele
- Arbeiten mit Funktionen
- Fehlerbehandlung mit try/except
- Kontrollfluss mit while-Schleifen
- Saubere Trennung von Zuständigkeiten
- Rückgabewerte und Abbruchlogik

## Funktionsweise
- Der Benutzer gibt an, wie viele Noten er erfassen möchte
- Es werden genau so viele **gültige** Noten abgefragt
- Ungültige Eingaben zählen nicht
- Abbruch ist jederzeit mit `0` möglich

## Hinweis
Dieses Projekt ist ein Lernprojekt und wird iterativ weiterentwickelt.
Der Fokus liegt auf Verständlichkeit und sauberer Struktur, nicht auf Perfektion.
